# 投稿指南

感谢你愿意给“测控补丁”添砖。这个仓库的目标不是把所有课都搬一遍，而是尽量补齐测控专业特色课程中资料稀缺、经验分散、信息不透明的部分。

如果嫌 GitHub 投稿太麻烦，也可以直接联系 QQ：`1031804897` 投稿。维护者会在整理后标注投稿来源；如果希望匿名，请在投稿时说明。

## 投稿前请确认

投稿内容应尽量满足以下条件：

- 与西安交通大学测控专业、仪器科学与技术学院或原电测控、机测控、食品测控方向有关；
- 主要面向测控特色课程、实验、培养方案、学习路线或生活经验；
- 不与已有大型资料库中的公共基础课资料重复；
- 不包含个人隐私、未授权传播内容或明显侵权风险；
- 实验代码和实验文档可以投稿，但应以理解、复现、排错和优化为目的，不应鼓励直接照搬提交。

## 推荐资料类型

- 复习提纲、知识框架、公式整理；
- 基于课堂 PPT 或教材整理出的复习资料；
- 往年题回忆、题型说明、考试范围变动记录；
- 实验说明、实验代码、实验文档、实验环境配置、数据处理脚本、常见问题和避坑经验；
- 课程评价、选课建议、培养方案解读；
- 与测控方向相关的公开资料索引。

## 实验材料投稿原则

本仓库允许收录实验代码和实验文档，是为了帮助后来者理解实验、复现实验流程、检查环境配置，并在已有基础上继续优化。请尽量把材料整理成“可学习、可修改、可追溯”的形式，而不是只放一个可以直接交的最终答案。

建议实验材料至少说明：

- 对应课程、实验名称和适用年份；
- 运行环境、依赖版本、硬件或软件平台；
- 代码或文档的来源，例如个人编写、同学投稿、AI 辅助生成、课堂材料整理；
- 已知问题、未完成部分和可以改进的方向；
- 是否删除了姓名、学号、实验成绩、教师批注等个人信息。

请不要将本仓库内容原样提交为自己的课程作业或实验报告。在抄袭之前，请先想一想：这个项目是公开的，老师、助教和同学都可能看到同一份材料。公开收录实验材料的初衷，是承认 AI 已经很强、大家也会用 AI 写代码和整理报告；既然如此，不如把可以公开的基底材料放出来，让大家在理解和优化的基础上继续往前走。

## 文件目录

建议按课程名称建立目录：

```text
课程名称/
  README.md
  复习资料/
  PPT整理/
  往年题/
  实验/
  经验/
  参考资料/
```

如果课程名称有多个版本，请优先使用培养方案或教务系统中的正式名称，并在课程 `README.md` 中补充别名。

## 命名建议

文件名建议包含课程、内容、年份或整理者信息，例如：

```text
光电检测技术/复习资料/光电检测技术-复习提纲-2026-匿名.pdf
智能仪器设计/经验/智能仪器设计-课程经验-2026-匿名.md
误差理论与数据处理/PPT整理/误差理论-第1-4章整理-2026-匿名.md
```

不强制实名。若选择匿名，可以使用“匿名”“某同学”等标识。

## AI 整理资料规范

如果资料由 AI 辅助生成或整理，请在文件开头或课程 `README.md` 中说明：

- 原始材料类型，例如课堂 PPT、教材、课堂笔记；
- AI 参与方式，例如提纲生成、错别字修正、公式整理、知识点归纳；
- 是否经过人工校对；
- 已知问题或不确定内容。

建议使用如下说明：

```text
本文由课程 PPT 经 AI 初步整理后人工校对完成。公式、定义和考试范围仍可能存在错误，请以课堂要求和教材为准。
```

## 隐私与版权

提交前请检查：

- 是否删除了姓名、学号、班级、手机号、邮箱、宿舍号等个人信息；
- 是否删除了实验报告封面、教师批注、成绩截图等敏感内容；
- 是否拥有分享该资料的权利，或资料本身适合公开传播；
- 是否避免上传完整教材扫描件、未授权课件、付费资料或受限平台内容。

如果某份资料不适合公开上传，可以考虑只写经验说明或资料索引。

## Pull Request 建议

提交 PR 时请尽量说明：

- 涉及课程；
- 资料类型；
- 适用年份或培养方案版本；
- 资料来源；
- 是否经过人工校对；
- 是否存在版权、隐私或准确性风险。

维护者可能会因为重复、范围不符、隐私风险、版权风险或学术诚信风险而要求修改或拒绝合并。
